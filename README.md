# KSD-Augmented Adjoint Schrödinger Bridge Sampler

Augmenting [ASBS](https://arxiv.org/abs/2506.22565) with a Kernel Stein Discrepancy (KSD) penalty and Density-Adaptive Regression Reweighting (DARW) to prevent mode collapse in sampling from Boltzmann distributions.
Forked from [facebookresearch/adjoint_samplers](https://github.com/facebookresearch/adjoint_samplers).
