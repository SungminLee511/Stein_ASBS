# SDR-ASBS: Stein Discrepancy Regularized Adjoint Schrödinger Bridge Sampler

Augmenting [ASBS](https://arxiv.org/abs/2506.22565) with SDR (Stein Discrepancy Regularization) — combining a KSD penalty with DARW reweighting to prevent mode collapse in sampling from Boltzmann distributions.
Forked from [facebookresearch/adjoint_samplers](https://github.com/facebookresearch/adjoint_samplers).
